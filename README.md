# Code Academy Frontend Project #1

## To run the project follow the steps below

## Clone de repository

```bash
git clone https://github.com/FabioJPC/code-academy-frontend-project-1.git
cd code-academy-frontend-project-1
```

## Build and run the container

```bash
docker compose up -d --build
```

## Acess the webpage

```bash
http://localhost:8080
```
